Mixins

            A mixin is a way of adding properties and methods of some objects to another object without using inheritance. 
            It allows more complex objects to be created by ‘mixing’ basic objects together
            for (const object of objects) {
            if(typeof object === 'object') {
                for (const key of Object.keys(object)) {
                    if (typeof object[key] === 'object') {
                        target[key] = Array.isArray(object[key]) ? [] : {};
                        mixin(target[key],object[key]);
                    } else {
                        Object.assign(target,object);
                    }
                }
                }
            }
            return target;
            }

            We can just mix in an object literal and add both properties at once:
            mixin(wonderWoman,{ name: 'Wonder Woman', realName: 'Diana Prince' });
            wonderWoman.change()

Chapter 15

Modular JavaScript - ES6

            Exporting Modules - There is more than one way to export modules in ES6.
            Include the keyword export in your function definition:
            export function setName(name) { ... }

            Include an export object with the properties you want to be included in the module.
            function addResult() { ... }
            function updateScoreboard() { ... }
            export { addResult, updateScoreboard };

            Import an entire module as scoreboard
            import * as scoreboard from ‘./scoreboard.js’;

            Import specific elements of a module
            import { getName as getPlayerName, logPlayer } from ‘./player.js’;
